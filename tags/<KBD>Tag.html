The KBD Tag <KBD>

Elements
Info

The <KBD> tag is used to format text. It formats the text between the tag into Keyboard Input. so it looks like this.


Attributes

The <KBD> tag supports all Global and Event attributes. There are a lot so I didn't list them all.

Example

Code


<html>
<head>
<title>
KBD Tags
</title>
</head>
<body>
<kbd> Keyboard Input </kbd>
<kbd> This is fun to type in. </kbd>
<p> Normally it would look like this. </p>
<kbd> But this is so much cooler!!!! </kbd>
</body>
</html>
