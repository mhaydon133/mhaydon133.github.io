The Header Tag <H4>

Elements
Info

The <H4> tag is used as a header format. It is not used very often as there are three other header format tags that come before it: <H1> - <H3>.


 Attributes

The <H4> tag supports the align attribute as well as the global and event attributes in HTML.

Example

Code


<html>
<head>
<title>
HTML Tags
</title>
</head>
<body>
<h1> Header 1 </h1>
<h2> Header 2 </h2>
<h3> Header 3 </h3>
<h4> Header 4- Our header </h4>
<h5> Header 5 </h5>
<h6> Header 6 </h6>
</body>
</html>
